# UNDER

UNDER is an exploratory intel-map project inspired by Palantir Gotham.

Inspiration:
- Palantir Gotham style workflows and UI concepts
- https://www.youtube.com/watch?v=rXvU7bPJ8n4

## Project Status

This is an exploratory prototype for experimentation, not a production system.

## Tech Stack

- React
- Vite
- MapLibre GL
- Three.js

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Data sources in this project are a mix of open/public feeds and exploratory integrations.
- Availability, freshness, and coverage depend on each external provider.
